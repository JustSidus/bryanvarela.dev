Sobre mí

Construyo entornos seguros y resuelvo problemas. Tengo una base lógica fuerte y mi trabajo está orientado totalmente a la nube. Mi fuerte no es memorizar sintaxis, sino analizar arquitecturas, investigar dependencias y diseñar soluciones.

Cómo trabajo

  • Investigación técnica: Analizo el requerimiento, evalúo los servicios disponibles y defino la arquitectura que mejor se adapte al caso de uso.
  • Adaptabilidad: Soy agnóstico al lenguaje. Si el backend requiere C#, Node.js o el frontend pide React, uso mi base lógica para adaptarme al stack sin perder velocidad.
  • Enfoque Cloud: Toda mi experiencia es nativa de la nube. Conecto y despliego servicios usando Azure, Google Cloud Run y Cloudflare.

Prioridades técnicas

  • Seguridad: Diseño bajo el principio de menor privilegio. Manejo aislamiento de red, gestión segura de secretos y auditoría de accesos.
  • Rendimiento: Optimizo el uso de recursos y el tiempo de respuesta en la nube para mantener los costos controlados.
  • Decisiones objetivas: La arquitectura dicta la herramienta. Justifico la adopción de cada tecnología basándome en su eficiencia para el proyecto.

Arelify Platform - SaaS B2B multi-tenant

Rol: Backend & Cloud Software Engineer, fundador técnico.

Stack: .NET 10, PostgreSQL 16, React 19, Cloudflare Workers, Google Cloud Run, WorkOS.

Estado: En producción, pilotos activos en República Dominicana.

SaaS multi-tenant que le da a cada PYME un subdominio propio indexable por Google, agendamiento 24/7 con prevención de doble reserva a nivel de base de datos y un editor visual para que el dueño actualice su sitio sin depender de un desarrollador. Reservas online para PYMEs que hoy operan exclusivamente por WhatsApp.

Arquitectura: Cloudflare Worker como BFF (resuelve slug del negocio, prerendera HTML+JSON-LD para SEO real). Frontend React 19 + Vite sobre Cloudflare Pages (un bundle, tres superficies: landing, portal público, panel admin). Backend .NET 10 sobre Google Cloud Run con PostgreSQL multi-tenant.

Visitor Management System

Rol: Full-stack engineer.

Stack: Vue 3, TypeScript, Laravel (PHP 8), MySQL, Azure.

Métricas: Tiempo de registro ~28 segundos, 3 roles (recepción / anfitrión / guardia), trazabilidad 100% QR.

VS Code Extension · paste-image-to-terminal

Rol: Developer tool creator.

Stack: VSCode API, TypeScript, SHA-256 deduplication, multi-plataforma.

Extensión que pega imágenes al terminal usando secuencias de escape bracketed paste. Deduplicación por SHA-256 para evitar duplicados, manejo de multi-plataforma (Windows/Mac/Linux) sin dependencias npm, y concurrencia segura. Uso personal y disponible en su repositorio público.
